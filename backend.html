Configuration Required

To enable this dashboard, you need to:

  1. Create a Supabase project at supabase.com
  2. Run the SQL schema from docs/SUPABASE_MIGRATION.md
  3. Add SUPABASE_URL and SUPABASE_ANON_KEY to GitHub Secrets
  4. Update tracker.js to send data to Supabase
