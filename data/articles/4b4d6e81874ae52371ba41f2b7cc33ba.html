Codex passe à la vitesse supérieure avec GPT‑5-Codex

Source: ActuIA | Date: 9/17/2025 | Lang: FR | Original Article

OpenAI a annoncé le lancement de GPT‑5-Codex, une version de GPT‑5 optimisée pour l'ingénierie logicielle agentique dans Codex, son agent IA lancé en mai dernier. Selon l'entreprise, ce modèle qui adapte son temps de réflexion de manière plus dynamique peut fonctionner de manière autonome pendant plus de 7 heures d'affilée sur une tâche complexe.

GPT‑5-Codex est pensé pour accompagner les développeurs dans toutes les étapes du cycle logiciel : création de projets, ajout de fonctionnalités, refactoring, débogage, revue de code et tests. Il peut être sollicité pour des tâches ponctuelles ou laissé en autonomie pour exécuter des missions longues, tout en maintenant un haut niveau de fiabilité.

Revue de code intelligente

GPT-5-Codex a été entraîné spécifiquement pour la revue de code et la détection de défauts critiques. Lors de ce processus, il parcourt la base de code, prend en compte les dépendances, exécute le programme ainsi que les tests associés afin d’en vérifier la validité.

OpenAI a évalué ses performances en s’appuyant sur des validations récentes de dépôts open source populaires. Chaque révision a été examinée par des ingénieurs logiciels expérimentés, chargés d’en évaluer la pertinence et l’exactitude. Ils ont constaté une nette amélioration par rapport aux versions précédentes : moins de commentaires incorrects et plus de suggestions efficaces.

Intégration dans l'environnement Codex

Codex est désormais un compagnon omniprésent pour les développeurs :

  • Terminal (Codex CLI) : interface modernisée, suivi des tâches, gestion des diffs et accès réseau configurable ;

  • IDE (VS Code, Cursor, etc.) : exploitation du contexte local, prompts simplifiés, synchronisation transparente avec le cloud ;

  • Web et GitHub : revue automatisée des pull requests, exécution des tests, intégration dans les workflows collaboratifs ;

  • Application iOS ChatGPT : continuité du développement en mobilité, avec accès direct aux fonctionnalités Codex.

Sécurité et contrôle

Codex fonctionne par défaut dans un environnement sécurisé (sandbox), avec des paramètres personnalisables selon le niveau de risque. Il demande des autorisations avant toute action sensible et fournit des logs, des citations et des résultats de test pour chaque tâche.

Il est d'ores et déjà inclus dans les plans ChatGPT Plus, Pro, Business, Edu et Enterprise. OpenAI, qui prévoit également de le rendre accessible via son API dans un avenir proche, déclare :

"Avec ces mises à jour, Codex se rapproche de ce que nous avons toujours construit : un coéquipier qui comprend votre contexte, travaille à vos côtés et prend en charge le travail de votre équipe de manière fiable."
