Should be solved now by upgrading to the latest beta.

The build servers had to be adjusted because of a big (&$!@) change by mosquitto's CMake project.

Domoticz is now also using a new external library (libmosquitto1) that is installed by the updatebeta/updaterelease scripts (assuming you are on one the latest beta's)

For users running a stable release via the bash install command line , you should run this script again so all dependencies are installed.

Of course docker users will have no issues when upgrading
