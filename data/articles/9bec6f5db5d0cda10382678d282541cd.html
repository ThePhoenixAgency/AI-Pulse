Official Ubuntu 26.04 ‘Resolute Raccoon’ mascot revealed

Source: OMG! Ubuntu | Date: 2/15/2026 8:40:35 PM | Lien | Lang: EN 
Official mascot artwork for Ubuntu 26.04 LTS ‘Resolute Raccoon’ has been revealed. As the upcoming release dubbed ‘Resolute Raccoon’, the monochromatic mascot naturally features its namesakes’ characteristic masked face and tail, albeit rendered in Ubuntu’s trademark geometric style. The tail is incorporated into a spiralling starburst. Dark 26.04 mascot artwork (left); light version (right) You can expect to see Ubuntu’s new mascot artwork on the new default wallpaper (due to be unveiled shortly), and glimpse it in the various promotional materials and social media posts Canonical and the wider Ubuntu project will put out as we approach April. As source code is made available, community creations can make use of it too. Some find Ubuntu’s animal-themed codenames charming others cringe, but most will agree that they’re an established part of the distro’s identity. A good mascot won’t make the OS any faster (nor a bad one slower), but a unique identity does make one release distinct from the rest. Ubuntu 26.04 LTS is released on April 23, 2026 as a long-term support release. The Resolute Raccoon follows October 2025’s Questing Quokka, and joins a menagerie of mascots variously spanning species with skills of all stripes, Artful Aardvarks to Karmic Koalas. Download the Ubuntu 26.04 mascot artwork as an SVG file below. You can import these files (light and dark) into an image editor like Inkscape to create your own Ubuntu artwork, e.g., custom wallpapers for your testing rig, phone, etc. Canonical will add the mascot to the Ubuntu Google Drive artwork folder in the next day or two.
