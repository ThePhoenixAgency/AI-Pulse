Firefox 147 has received yet another bug-fixing update to address bugs with the new tab page and a single security issue. Taras Buria Neowin @TarasBuria · Feb 16, 2026 11:32 EST Mozilla has released one more update for Firefox 147, the latest feature update for the browser that arrived last month. It is the fourth minor update for version 147, and it includes two fixes to address two issues: one with the new tab page and a single security issue. Firefox 147.0.4 is now available for download from the official website and within existing installations. The official release notes mention only one bug related to the new tag page. Mozilla fixed a problem with the new tab page appearing blank instead of showing the typical stuff like your pinned websites, the search bar, recent pages, and more: Fixed an issue that could cause the New Tab Page to appear blank for some users. (Bug 2014616) In addition to fixing the broken new tab page, Mozilla resolved a security vulnerability that allowed heap buffer overflow in libvpx (CVE-202602447). This patch is available not just in Firefox 147.0.4, but also in Firefox 140.7.1 ESR and Firefox 115.32.1 ESR. Firefox 115 ESR remains the only officially supported Firefox version for those who still use Windows 7, Windows 8, and Windows 8.1. At this point, Mozilla is the only mainstream browser that still provides security patches for its browser on long-outdated Windows versions. Right now, the company has no specific dates for the end of Firefox 115 ESR. As for Windows 10, which reached the end of mainstream support last year, it is too early to talk about the end of browser support for it, as it has eight months of extended support ahead of it, plus several more years of paid updates for business customers. You can download Firefox 147.0.4 by opening Menu > Help > About Firefox. Alternatively, get the browser from the official website or the Microsoft Store.
