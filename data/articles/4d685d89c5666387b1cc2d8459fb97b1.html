Agent Multiplexer

Run dozens of Claude Code agents in parallel — monitored, controlled, and coordinated from a single dashboard. Access everything from your browser, phone, or terminal.

  • Parallel agents — run any number of Claude Code sessions simultaneously, each isolated in its own tmux pane with its own conversation history. Start, stop, and send commands to any agent without switching terminals.

  • Live status detection — every session card shows whether an agent is working, needs input, or idle, updated in real time via Server-Sent Events. Know at a glance which agents need your attention without reading their output.

  • Web dashboard + PWA — full-featured HTTPS dashboard installable as a home screen app on iOS and Android. Monitor agents, send commands, and browse output from your phone. Offline-capable with queued command replay on reconnect.

  • Multi-pane grid view — switch to a full-screen tiled layout to watch multiple agents' live output side by side. Each pane has its own send bar. Drag to rearrange, and positions are saved across sessions.

  • Live peek & control — scroll through any agent's full scrollback (including stopped sessions, which are snapshotted to disk), search output, send commands, and use quick-action chips (Ctrl-C, /compact, /clear) — all without attaching to the terminal.

  • File attachments — send files directly to any agent. Paste an image with Ctrl-V, drag and drop onto the send bar, or click the button. Supports images, PDFs, text, CSV, JSON, and more.

  • Session management — rename, duplicate, clone & continue (fork a conversation into a new session), switch models mid-session, toggle YOLO mode, pin sessions to the top, and adopt existing tmux sessions not created by amux.

  • Built-in kanban board — track work across all agents in one place. Auto-generated issue keys (PROJ-1), tags, due dates, drag-and-drop columns, calendar view, and an iCal feed for Google Calendar. Backed by REST API.

  • Tailscale & remote access — the server auto-provisions TLS certificates (via Tailscale, mkcert, or self-signed fallback). Access your agents securely from your phone over Tailscale — no VPN config, no open ports.

  • REST API — every action is an endpoint. Automate orchestration with curl or any HTTP client. Register, start, stop, send, query status, and manage board items programmatically.

  • Session & global memory — each agent gets its own persistent memory file, plus a shared global memory. Edit directly in the dashboard and inject context into any session.

  • Token stats — daily token usage across every session with per-agent breakdowns and bar charts. Know exactly what each agent is spending before your bill arrives.

  • Zero dependencies — single Python file, no build step. Requires only Python 3 and tmux. Install in seconds, run anywhere.
