Kerbal Space Program spiritual successor Kitten Space Agency now has a Linux version

Source: GamingOnLinux | Date: 2/23/2026 11:35:00 AM | Lien | Lang: EN

Kitten Space Agency is a promising upcoming space simulation game that's pretty much a spiritual successor to Kerbal Space Program and now for Linux too. It's still very early days for the Linux builds, and for the game as a whole.

Writing on the official forum, founder Dean Hall of RocketWerkz said:

Apologies it has taken this long, but we are now trialing linux build deployments. These builds are deployed untested. I have changed ⁠version-history on discord to now split out preparing the builds and deploying them. The windows build will now refuse to deploy a build that fails digital signature. Both builds will now indicate if they have successfully be deployed or not. The master server MAY indicate an incorrect build for linux, if linux fails to deploy. This will be changed soon when I can update that side of the deployment process.

You can download the trial builds for linux directly at: https://ksa-linux.ahwoo.com/

Hall also noted that currently their development team are burning through money as they're "spending x10 as much as we get in contributions", and suggested that if you contribute to make sure you note in a message that it's for the Linux build as they will "use this data to help push that we should continue to integrate this going forward".

The Linux build is a little on the rough side right now, with workarounds likely needed with comments on the forum post pointing out some of the issues.

See more on their Ahwoo distribution website. It's like a crowdfunding platform, with the game having currently received $136,546 from backers.
