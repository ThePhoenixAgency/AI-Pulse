Windows 11 intègre un outil pratique dans la barre des tâches

Source: GNT | Date: 2/19/2026 3:50:01 PM | Lien | Lang: FR

Publié le 19 février 2026 à 16:50 par Jérôme G.

Un outil de test de la connexion à Internet s'ajoute à la barre des tâches de Windows 11. La roue n'est toutefois pas réinventée avec ce qui s'avère être un simple raccourci.

Dans le cadre des builds 26100.7918 (Windows 11 24H2) et 26200.7918 (Windows 11 25H2) de Windows 11 qui sont diffusées dans le canal Release Preview pour les Windows Insiders, Microsoft intègre un outil de mesure de la performance de la connexion Internet. C'est la confirmation de la trouvaille qui avait été faite l'année dernière par le leaker Phantomofearth.

The network icon in the system tray's context menu and the Wi-Fi quick settings page in Windows 11 are getting buttons to let you quickly run a network speed test. (Takes you to Bing to do the speed test.) pic.twitter.com/ZXcQvs5BP8

Comment fonctionne ce nouvel outil ?

Directement depuis la barre des tâches de Windows 11, l'outil est accessible via les Paramètres rapides Wi‑Fi ou Cellulaire, ou en faisant un clic droit sur l'icône réseau dans la zone de notification. Une option est ensuite proposée pour effectuer un test de débit.

Il ne s'agit cependant pas d'un outil entièrement natif intégré à Windows. En réalité, le système ouvre simplement le navigateur web de l'utilisateur pour effectuer le test. La mesure peut évaluer les performances des connexions Ethernet, Wi-Fi et cellulaires.

" Cette fonctionnalité permet de vérifier les performances du réseau et de résoudre les problèmes ", écrit Microsoft.

S'agit-il d'un service de test propriétaire de Microsoft ?

Microsoft ne lance pas un concurrent direct aux services établis du type Speedtest.com ou Fast.com. L'année dernière, les premières observations avaient montré que le raccourci redirigeait vers le moteur de recherche Bing.

Bing utilisait alors la technologie de Speedtest.com d'Ookla pour fournir les résultats. Microsoft confirme qu'une approche similaire est maintenue, et donc par le biais du navigateur web par défaut.

L'objectif est ainsi d'offrir un raccourci pratique, pas de créer un service de mesure indépendant.
