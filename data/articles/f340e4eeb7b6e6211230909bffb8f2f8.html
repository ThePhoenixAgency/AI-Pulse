It's now easier to install MGSHDFix for Metal Gear games on Linux / Steam Deck

Source: GamingOnLinux | Date: 2/20/2026 11:25:04 AM | Lien | Lang: EN

The ace Luxtorpeda project has expanded, and now it makes it real easy to install the popular MGSHDFix pack for various Metal Gear games on Linux / Steam Deck.

It works in a similar way to Proton / GE-Proton. You add Luxtorpeda as a Compatibility Tool inside Steam using something like ProtonUp-Qt, and then set specific games to run with it in their properties. What it enables is access to many different game engines like OpenMW for Morrowind, openXcom for X-COM: UFO Defense, OldUnreal for Unreal Tournament and so many more.

Over the past few updates though, Luxtorpeda has also expanded to include running popular patches and mod packs for certain Windows games with Proton - with it doing all of the work for you. All you need to do is just set the games to use Luxtorpeda.

This includes a new recent addition of the Metal Gear / Metal Gear Solid games to use the very popular MGSHDFix and MGSM2Fix packs that add in tons of quality of life improvements to the likes of Metal Gear 1/2 (MSX), Metal Gear Solid 1, Metal Gear Solid 2 and Metal Gear Solid 3.

Luxtorpeda is a fantastic project you should be keeping an eye on.
