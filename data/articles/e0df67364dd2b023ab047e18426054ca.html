I Built an Open-Source Guitar Tuner That Runs Entirely in Your Browser (No Libraries, No Backend)
Most guitar tuners on the web are black boxes.
You don’t know how they work.
You can’t modify them.
And almost none of them are built with accessibility in mind.
So I built my own.
It’s called PitchCraft.
An open-source chromatic tuner that runs entirely in your browser using the Web Audio API.
No backend.
No tracking.
No external tuning libraries.
Just signal processing and math.

I opened a popular web tuner and thought:
Why does this feel like magic I can’t see inside?
So I asked myself:
What if I build one completely from scratch?

That question became PitchCraft.
The Crazy Part: The Browser Is Enough
Modern browsers can do real-time pitch detection.
Here’s the pipeline:
Microphone → Frequency → Note → Visual Feedback
Using:
getUserMedia
AnalyserNode
autocorrelation
equal temperament math
That’s it.
No server needed.
The Core: Autocorrelation Pitch Detection
The tuner works by detecting periodicity in the waveform.
Simplified:
The algorithm compares the signal with delayed versions of itself.
When it matches, that delay equals the fundamental frequency.
From there:
Frequency → Note → Cents deviation
All in real time.
This is the same principle used in professional tuners.

What I Learned Building This
Three things surprised me:

  1. The Web Audio API is insanely powerful You can build real music tools in the browser.
  2. Signal processing is not as scary as it seems Basic math gets you far.
  3. Accessibility is not hard — but you must care Most tools simply don’t.
