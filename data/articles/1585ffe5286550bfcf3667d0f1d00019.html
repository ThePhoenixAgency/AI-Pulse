I do not remember a lot of it, but you may need to configure CAN EZ publish interval, so in case if you have a value which doesn’t change for long time, it will be posted from EZ3 memory to bus periodically. As far I remember CMI is just broadcasting values through CoE bridge, if it does not appear on CAN side it doesn’t get over CMI.
