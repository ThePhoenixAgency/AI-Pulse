iOS 26.3 has fixes for 35+ security issues on iPhone, details here

Source: 9to5Mac | Date: 2/11/2026 7:52:01 PM | Lang: EN | Original Article

Apple’s latest wave of software updates is lighter than usual on new features, but heavy on bug fixes and security updates, with 35+ security fixes included in iOS 26.3 and more.

Apple publishes details on security updates in latest OS releases

Following the release of iOS 26.3, iOS 18.7.5, macOS Tahoe 26.3, and more, Apple has published the full list of security fixes included in the updates.

Here are links to the full security release notes for each new update:

  • iOS 26.3 and iPadOS 26.3
  • iOS 18.7.5 and iPadOS 18.7.5
  • macOS Tahoe 26.3
  • macOS Sequoia 15.7.4
  • macOS Sonoma 14.8.4
  • tvOS 26.3
  • watchOS 26.3
  • visionOS 26.3

On iPhone and iPad in particular, the list of fixes is especially long. By my count there are 37 security issues addressed by iOS and iPadOS 26.3.

Most of these don’t necessarily have evidence of being exploited. However, there’s at least one exception.

In the case of a dyld vulnerability where an “attacker with memory write capability may be able to execute arbitrary code,” Apple says it’s “aware of a report that this issue may have been exploited in an extremely sophisticated attack against specific targeted individuals on versions of iOS before iOS 26.”

Read the full list of security updates and fixes via the links above.
