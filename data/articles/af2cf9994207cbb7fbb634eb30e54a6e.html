Every time I start up Zwave JS UI, I get a “new version available” popup for v11.xx. I’m currently on latest v7.0.0. If I hover over the info bubble in the upper right it says I’m running v11.x. But if I look at the “Settings > Apps > Zwave JS UI” it says v7.x. If I check updates within HASS or the ZWAVE JS UI app interface, I get no updates available.

What am I missing here?
Why the inconsistencies?
How do I upgrade to the latest v11.x if needed?




You’re using an app, it means you get upgrades to ZUI when the app itself is updated. You don’t have the option of upgrading yourself.

Z-Wave JS - Which version? summarizes the various options.

You can turn off the notification in ZUI Settings.
