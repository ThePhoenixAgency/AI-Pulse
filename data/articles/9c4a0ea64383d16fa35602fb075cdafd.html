Meishi défie Apple/Google: Contacts P2P E2EE Open Source

Source: Journal du Hacker | Date: 2/17/2026 10:00:18 AM | Lien | Lang: FR

Le problème Combien de copies de vos coordonnées existent actuellement ? Qui les détient ? Comment sont-elles utilisées ? Nous avons perdu le contrôle de nos données, ou plutôt, nous ne l’avons jamais eu. Les géants de la technologie contrôlent des milliards de bases de données de contacts. Les gouvernements développent des identités numériques centralisées (EUDI Wallet EU en 2026). Nous subissons : violations de données, spam, absence totale de contrôle.

La solution : architecture P2P + E2EE Meishi renverse le paradigme. Lorsque vous mettez à jour votre profil numérique, tous vos contacts reçoivent automatiquement la mise à jour. Vous gardez le contrôle : vous accordez, modifiez ou révoquez l’accès quand vous le souhaitez.

Stack technique

  • P2P + E2EE avec Signal Protocol pour les données privées
  • Zero-knowledge : serveurs sans accès aux données privées
  • Open source (moteur de synchronisation + client iOS/Android)
  • Hybride : client-serveur uniquement pour les données publiques, P2P pour le reste
  • Confidentialité dès la conception : pas de suivi, pas de vente de données
  • Synchronisation automatique : mises à jour automatiques pour tous les contacts
  • Contrôle total : accorder/révoquer l’accès en temps réel
  • Conforme au RGPD : API avec opt-in explicite
  • Open source : moteur de synchronisation et application client vérifiables

Le manifeste : https://github.com/Meishiapp/docs/blob/main/Meishi_Manifesto_EN.md Extrait de notre manifeste :

« Celui qui contrôle les données commande. La valeur des données ne réside pas dans leur possession, mais dans leur véracité, et seul leur propriétaire légitime peut garantir leur authenticité grâce à un contrôle total. »

Ce projet n’est pas destiné à tout le monde. Il s’adresse à ceux qui souhaitent reprendre le contrôle de leur identité numérique.
