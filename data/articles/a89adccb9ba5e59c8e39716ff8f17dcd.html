CoSave - Save 95% on Claude & Gemini Costs

What is CoSave?

CoSave is a VSCode extension that dramatically reduces your AI coding costs by up to 95% through intelligent dual-model optimization, while enabling unattended sequential task execution.

The Philosophy

CoSave = Save 95% on Costs + Long Memory + Sequential Tasks + Work from Bed

Features

  • 95% Cost Reduction: Intelligent dual-model system uses small parameter models (e.g., Gemini Flash) for reading/analysis and Claude/Gemini large models only for code generation
  • Long Memory: Automatically accumulates project development experience and conventions in mem.md, enabling the AI to better understand your project over time
  • Sequential Task Execution: Set up 100+ tasks before bed, click execute, and wake up to completed work
  • Remote Control: Monitor and manage tasks from your phone anywhere, anytime
  • Multi-Task Parallel Work: Process multiple parallel tasks simultaneously, significantly reducing wait time

Get Started

CoSave has "dual model mode" enabled by default, no additional setup required to start using it.

  1. Install the extension
  2. Configure your AI model settings
  3. Create your task sequence
  4. Click execute and let CoSave do the work!
