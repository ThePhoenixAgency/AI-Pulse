visionOS 26.4 Beta 1 Now Available for Apple Vision Pro Developers

Source: The Mac Observer | Date: 2/16/2026 7:13:19 PM | Lang: EN | Original Article

Apple has started the visionOS 26.4 beta cycle with the first developer build now available for Apple Vision Pro. The update landed on February 16, 2026, and it carries build number 23O5209m.

This arrives just days after the public release of visionOS 26.3 on February 11, 2026 (build 23N620), which effectively closed out the 26.3 release cycle before this new round of testing began.

As usual with the first beta in a new point release, the early focus tends to be stability work, under-the-hood fixes, and smaller feature changes that show up as developers spend time with the build.

How to update to visionOS 26.4 beta 1

  • Back up your Apple Vision Pro before you install beta software.
  • On Apple Vision Pro, open Settings.
  • Tap General.
  • Tap Software Update.
  • If you see Beta Updates, select it and choose the visionOS Developer Beta option, then return to Software Update.
  • Tap Download and Install (or Install Now) and follow the prompts.

During installation, the headset will ask you to take it off. You can track progress on the front display with EyeSight.

Changes in visionOS 26.4 beta 1

Apple has posted the developer beta as visionOS 26.4 beta (23O5209m), but there is not yet a clear public list of headline additions tied to this first build.

Here is what you should watch for as testing begins:

  • Bug fixes and performance changes that are not obvious until you use the headset for a few days.
  • App compatibility updates, especially for system apps and developer tools workflows.
  • Any movement on Siri and Apple Intelligence features, including changes to on-device behavior, prompts, or system surfaces where those features appear.

We are still digging through the beta, and we will report back if we spot any major changes, including anything tied to Siri and Apple Intelligence.
