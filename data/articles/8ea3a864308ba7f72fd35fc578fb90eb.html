La fin des amitiés brisées : WhatsApp s'attaque enfin au fléau des spoilers

Source: GNT | Date: 2/23/2026 9:35:01 AM | Lien | Lang: FR

Publié le 23 février 2026 à 10:35 par Mathieu M.

WhatsApp s'apprête à déployer une nouvelle option de formatage pour masquer les révélations indésirées. Repérée dans les versions bêta, cette fonctionnalité permettra de cacher un texte derrière un bandeau, protégeant ainsi les utilisateurs des spoilers sur leurs séries ou films préférés. Une simple pression suffira pour dévoiler le contenu.

WhatsApp, la messagerie de Meta, est sur le point de résoudre un problème majeur des discussions de groupe : les spoilers. Des indices découverts dans les récentes versions bêta de l'application, sur iOS comme sur Android, révèlent le développement d'une fonctionnalité permettant de masquer une partie d'un message. L'objectif est simple : permettre à chacun de participer aux conversations sans craindre de se faire gâcher la fin de sa série favorite.

Comment cette fonction anti-spoiler fonctionnera-t-elle concrètement ?

Pour masquer un message, l'utilisateur aura le choix entre deux méthodes distinctes. La première, la plus intuitive, passera par le menu de formatage contextuel, où une nouvelle option Spoiler rejoindra les traditionnels Gras et Italique. Cette approche est similaire à ce que proposent déjà des plateformes comme Reddit.

La seconde méthode, plus rapide pour les habitués, s'appuiera sur un raccourci syntaxique. Il suffira d'encadrer le texte à dissimuler par des doubles barres verticales (||comme ceci||) pour que le formatage s'applique automatiquement. Une fois envoyé, le texte apparaîtra flouté ou recouvert d'un bandeau gris. Une simple pression du destinataire révélera le contenu masqué. Pour l'instant, cette option semble limitée aux messages texte, sans confirmation pour les images ou vidéos.

Quel est le véritable intérêt pour les discussions de groupe ?

Le principal champ de bataille contre les spoilers, ce sont les discussions de groupe. Que ce soit pour une série, un film ou un événement sportif, les participants n'ont pas toujours les mêmes rythmes de visionnage. Cette nouvelle fonctionnalité permettra d'éviter les frustrations et les tensions, en laissant à chacun le contrôle sur ce qu'il découvre. Fini le besoin de quitter temporairement un groupe ou de couper les notifications en attendant d'être à jour.

Cette nouveauté s'inscrit dans une tendance plus large chez WhatsApp visant à améliorer l'expérience utilisateur dans les conversations à plusieurs. Récemment, l'application a par exemple déployé une option pour que les nouveaux membres puissent voir une partie de l'historique d'un groupe. L'idée est de rendre les discussions plus saines et de donner un contrôle accru aux utilisateurs.

À quand le déploiement de cette nouveauté ?

Pour le moment, il n'y a aucune date de sortie officielle communiquée par Meta. La fonctionnalité est en développement actif, comme l'a repéré le site spécialisé WABetaInfo dans les versions TestFlight pour iOS et les bêtas Android. Le fait qu'elle soit développée simultanément sur les deux systèmes d'exploitation majeurs est un signe encourageant, suggérant un futur déploiement multiplateforme.

Cependant, il faut rester prudent. De nombreuses fonctionnalités testées dans les versions de test n'atteignent jamais le grand public. Dans ce cas précis, la fonctionnalité, bien que visible dans le code, n'est pas encore activée et ne fonctionne pas pour les messages envoyés. Si les tests sont concluants, on peut espérer un déploiement dans les quelques semaines ou mois à venir via une mise à jour standard de l'application.

Foire Aux Questions (FAQ)

Cette fonctionnalité s'appliquera-t-elle aussi aux images et vidéos ?

Pour l'instant, les informations disponibles indiquent que la fonction de masquage des spoilers ne concernera que les messages texte. WABetaInfo spécule qu'un support pour les médias comme les images ou les vidéos pourrait être ajouté ultérieurement, mais rien n'est confirmé par WhatsApp à ce stade.

Le message reste-t-il masqué après une première lecture ?

Les rapports divergent légèrement, mais la plupart suggèrent qu'une fois que l'utilisateur a tapé sur le message pour le révéler, il reste visible pour cet utilisateur dans la conversation. Le but est d'éviter une révélation accidentelle, pas de devoir interagir avec le message à chaque ouverture de la discussion.

Cette fonctionnalité est-elle déjà disponible pour les bêta-testeurs ?

Non. Bien que des références à la fonctionnalité aient été trouvées dans le code des versions bêta, elle n'est pas encore active. Les utilisateurs inscrits au programme bêta ne peuvent donc pas encore l'utiliser. Elle est toujours considérée comme étant en cours de développement interne.
