So, What's in The Bundle?

The 15 books here span a good range of topics. Here's a breakdown of what's included.

Linux administration is covered from a few different angles. Efficient Linux at the Command Line focuses on boosting productivity at the shell, which is the kind of thing that pays off every single day.

Learning Modern Linux takes a broader look at Linux as it's actually used today, including file permissions, application dependency management, and networking. Practical Linux System Administration covers the operational side of managing users, storage, setting up SELinux, networking, and services on live systems.

The Linux Pocket Guide rounds it out as a handy reference for common commands and tasks.

On the DevOps and automation side, Python for DevOps covers using Python for scripting, automation, and working with infrastructure APIs, which has become a core skill in most DevOps roles.

Ansible: Up and Running is one of the better introductions to configuration management and automation with Ansible, now in its third edition. Learning GitHub Actions gets into CI/CD workflows using GitHub's built-in automation platform.

For infrastructure as code, Terraform: Up and Running is widely considered one of the go-to books on provisioning cloud infrastructure. The third edition covers current Terraform practices, including modules, state management, and working across multiple providers.

The containers and orchestration section is well represented. Docker: Up & Running covers building and running containers from the ground up. Kubernetes: Up and Running handles orchestration, now in its third edition, with content on workloads, networking, and cluster management.

Security gets two dedicated titles. Learning DevSecOps focuses on integrating security practices into the development and deployment pipeline rather than treating it as an afterthought. Web Application Security covers offensive and defensive techniques for securing web applications, with the second edition splitting the curriculum into three parts.

Rounding out the bundle, Network Programmability and Automation is aimed at network engineers looking to apply DevOps principles to network infrastructure using Python and tools like Ansible.

FastAPI covers building APIs with the Python FastAPI framework, increasingly popular in backend and microservices work. And Learning Git is a solid foundation for version control, covering branching, merging, rebasing, and working with remote repositories.
