How it works

Chat

You talk to an AI agent. Residue quietly tracks the session.

Commit

A git hook links pending conversations to the commit SHA.

Push

On git push, sessions upload to your self-hosted worker.

Browse

See every conversation alongside the commits it produced.

Setup

1 Create an R2 bucket + S3 API token Details

4 Install the CLI

bun add -g @residue/cli

5 Connect to your worker

residue login

6 Set up a repo

residue init && residue setup <agent>

Full instructions including R2 setup, AI Search, and all options in the setup guide.

Open source and self-hosted

Deploy your own Cloudflare Worker. Your conversations stay on your infrastructure.
