Env Generator

A lightweight utility for generating and managing environment variable files (.env) from the command line.

Install

cargo install env-gen

Usage

env-gen

This will prompt you to create a new configuration file, optionally using an existing template file, and add key-value pairs for your environment variables. The generated 

.env
file will be created in the current working directory.

The following template files are supported (searched in the current working > directory):

  • example.env
  • .example.env
  • env.example
  • .env.example
  • env.sample
  • .env.sample
  • .env-dist
